rem vs em

by Artem Trgtsin

The question is, what is the best or indeed the most useful?

  • EM use the containers font size to calculate
  • REM use the HTML font-size as the base to calculate

Example

			<div class="container"> 
			    <p class="first"> 
					First Paragraph using EM's 
				</p>
			    <p class="second"> 
			  		Second Paragraph using REM's 
				</p>
			</div>

CSS

			html { font-size: 16px;}
			.container { font-size: 1.4em;}
			p.first { font-size: 1.4em;}
			p.second { font-size: 1.4rem;}

Result

First Paragraph using EM's

Second Paragraph using REM's

LESS mixins

			html { font-size: 62.5%;}
			 
			.font-size(@sizeValue) {
				@remValue: @sizeValue;
				@pxValue: (@sizeValue * 10);
				font-size: ~"@{pxValue}px"; 
				font-size: ~"@{remValue}rem";
			}

Usage

			p {
				.font-size(13); 
			}

The nice LESS mixins

			html { font-size: 62.5%;}
			.fontName (@size) { 
				@remValue: @size;
				@pxValue: (@size * 10);
				font: ~"@{pxValue}px" fontName, sans-serif; 
				font-size: ~"@{pxValue}px";
				font: ~"@{remValue}rem" fontName, sans-serif;
			}

Usage

			p {
				.fontName(18); 
			}
			p {
				font: 18px fontName, sans-serif;
				font-size: 18px;
				font: 1.8rem fontName, sans-serif;
			}
